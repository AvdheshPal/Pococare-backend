# Pococare-backend
 
## How to start;
1. First of all run this ```npm i``` commond in terminal.
2. Then create a file called .env in root folder and paste it ```DB = 'mongodb://127.0.0.1:27017'``` inside the .env file.
3. Then run ```npm start``` to start the server.
